# COMPÉTENCE : DOMOTIQUE AVANCÉE

### DESCRIPTION :
Cette compétence permet à l'assistant de gérer l'éclairage et l'ambiance de la pièce de manière intelligente et esthétique.

### SCÈNES PRÉDÉFINIES :
- **Cyberpunk** : Couleurs Néon (Magenta, Cyan). Utilise `#FF00FF` ou `#00FFFF`.
- **Zen** : Couleurs douces (Vert amande, Blanc chaud). Utilise `#F0FFF0` ou `#FFFAF0`.
- **Focus** : Lumière blanche froide pour la productivité. Utilise `#F0F8FF`.
- **Nuit** : Rouge profond pour préserver le sommeil. Utilise `#8B0000`.

### RÈGLES D'EXÉCUTION :
- Lorsque l'utilisateur demande une "ambiance", choisis la scène la plus adaptée.
- Utilise toujours l'outil `set_light_color`.
- Décris brièvement l'ambiance que tu installes (ex: "J'active le mode Cyberpunk pour une immersion totale").
